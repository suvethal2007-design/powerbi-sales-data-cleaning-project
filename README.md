📊 Sales Data Cleaning Project using Power BI
Project Overview

This project focuses on cleaning and transforming raw sales data using Power BI and Power Query. The objective was to improve data quality and prepare the dataset for future analysis and dashboard development.

Tasks Performed
Imported raw sales data into Power BI
Handled missing values
Removed duplicate records
Corrected data types
Standardized data formatting
Created a Sales_Category column
Validated data quality and consistency
Tools Used
Power BI
Power Query
Microsoft Excel
Skills Demonstrated
Data Cleaning
Data Transformation
Data Preparation
Power Query
Data Quality Management
Project Outcome

The dataset was successfully cleaned and transformed into a structured format, making it ready for analysis, reporting, and dashboard creation.
